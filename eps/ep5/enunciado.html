Para esta atividade, é recomendável se familiarizar com o formato de imagens PPM no modo texto (P3). O objetivo da atividade é implementar um leitor de imagens (coloridas PPM). Para isso, você vai implementar a função load_image_ppm do código disponível aqui. Este código executa a leitura de uma imagem com nome input.ppm e grava uma cópia com nome output.ppm. Observações: A função traz, na forma de comentário, a sequência de passos esperada para implementação. Você deve alterar SOMENTE a implementação do corpo da função. A estrutura do código e o tipo proposta para armazenamento de imagens devem ser mantidos inalterados. Envio: 05 de abril.
